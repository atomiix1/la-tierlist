Índice de Dificultad (DI): Combinación ponderada de componentes normalizados con z-score
IQR_rel (Z-normalizado): (Q3 - Q1) / Mediana - Variabilidad relativa
Skill Gap SG (Z-normalizado): Max / Mediana - Diferencia de rendimiento máximo
Ratio UpperWhisker RUW (Z-normalizado): (UpperWhisker - Mediana) / Mediana
DI: 0.5 × Z(IQR_rel) + 0.3 × Z(SG) + 0.2 × Z(RUW)
DI Final: Promedio ponderado por número de intentos (count) en cada puerta, solo usando datos que cumplen el mínimo de intentos
Tiers: S (DI ≥ +1.5σ) | A (+1.0σ ≤ DI < +1.5σ) | B (-0.5σ ≤ DI < +1.0σ) | C (-1.0σ ≤ DI < -0.5σ) | D (DI < -1.0σ)
Mayor DI = Mayor dificultad en la especialización | Los valores tachados no cumplen el mínimo de intentos requerido
